# Вивчай — Free Ukrainian Web Development Course

[vivchai.com.ua](https://vivchai.com.ua/)

A free, structured web development educational platform teaching programming in Ukrainian — from beginner to production-ready skills.

## What Is This

Вивчай (Vyvchai — "Learn" in Ukrainian) is a full-stack web development curriculum delivered entirely in Ukrainian. It covers the modern frontend stack in a practical, conversational style with real homework assignments and AI-assisted feedback.

## Curriculum

9 sequential learning blocks:

| # | Block | Topics |
|---|-------|--------|
| 1 | Tools & Environment | Git, GitHub, Node.js, npm, VS Code |
| 2 | HTML | Semantics, structure, forms |
| 3 | CSS | Layouts, Flexbox, Grid, animations |
| 4 | JavaScript (basics) | Types, control flow, functions |
| 5 | JavaScript (advanced) | ES6+, async/await, closures |
| 6 | TypeScript | Types, interfaces, generics |
| 7 | React | Components, hooks, state management |
| 8 | Next.js | SSR, routing, deployment |
| 9 | Practice | Projects and portfolio building |

## Features

- **26 graded homework assignments** — submit a GitHub repo link, get automated AI feedback
- **Built-in AI assistant** — context-aware help during lessons
- **Progress tracking** — via Google authentication
- **GitHub integration** — assignments build a real portfolio
- **Engaging content** — analogies, humor, historical context alongside technical depth

## Tech Stack

Built with Next.js, server-side rendering for lessons, client-side interactivity, theme switching, and search.

## Getting Started

Visit [vivchai.com.ua](https://vivchai.com.ua/) — no installation required. Sign in with Google to track progress and submit homework.
